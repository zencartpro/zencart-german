Überblick

Dieses Modul integriert den Lightbox Effekt in Zen-Cart. Alle großen Artikelbilder (auf der Detailseite) werden beim Anclicken in einer Lightbox angezeigt. Dadurch werden keine Popup Fenster mehr verwendet, was die Website wesentlich benutzerfreundlicher macht. Die zusätzlichen Artikelbilder werden innerhalb der Lightbox in einer Galerie angezeigt.

Mit Version 1.5.1 wird der Lightbox Effekt auf allen Seiten des Shops ermöglicht (auch auf EZ-Pages) und die Steuerung der Lightbox kann auch über Tastaturbefehle erfolgen. Der Code wurde optimiert und die Größe des Javascript-Codes um 52,23% reduziert.

Danke an Christophe Beyls für das Javascript Lightbox JS v2, Chris Boylan für die Arbeit am Image Handler 2 Support und an das Zen Cart Team für das beste verfügbare Open Source Webshop-System!

Neuinstallation

Diese Zen Lightbox Version ist nur für eine deutsche Zen Cart 1.3.8a Version geeignet!
Sollte im Shop bereits eine ältere Lightbox Version (z.B. 1.4) aktiv sein, muss die alte Version erst komplett entfernt werden! (siehe Update)

Denke daran, ein Backup von Website und Datenbank zu machen, bevor Du weiterliest.

Um die Lightbox zu installieren, gehe folgendermaßen vor:

  1. Benenne im includes Verzeichnis dieses Downloads folgende Ordner von DEINTEMPLATE auf das im Shop aktive Template um (z.B. classic, viennablood, ...)

    includes\languages\english\extra_definitions\DEINTEMPLATE
    includes\languages\german\extra_definitions\DEINTEMPLATE
    includes\modules\DEINTEMPLATE
    includes\templates\DEINTEMPLATE

  2. Lade alle Dateien und Ordner aus dem includes Ordner in der vorgebenen Struktur in den includes Ordner Deines Shops am Server. Es wird konsequent das Override-System verwendet, daher werden keine bestehenden Dateien überschrieben.

  3. Öffne im Ordner SQL dieses Downloads die install.sql mit einem Texteditor und kopiere den Text ins Eingabefeld unter Tools > SQL Patches installieren in Deiner Zen-Cart Administration. Durch Absenden werden die nötigen Datenbankeinträge angelegt.

EZ-Pages

Standardmäßig ist der Lightbox Effekt für alle Links zu Bildern innerhalb von EZ-Pages bereits aktiv. In der Lightbox-Konfiguration kann das ausgeschaltet werden.
Wenn Du also ein Bild mit Link zum größeren Bild so in eine EZ-Page einbaust:

<a href="images/bildgross.jpg"><img src="images/bildklein.jpg"></a>

Dann wird das Bild beim Anclicken in einer Lightbox vergrößert.

Ebenso geht das Bild in einer Lightbox auf, wenn nur ein Link zum Bild gesetzt wird und der Link angecklickt wird:

<a href="images/bildgross.jpg">Bild</a>

Werden innerhalb eines übergeordneten Elements mehrere Links zu Bildern gesetzt, werden all diese Bilder in der Galerieansicht dargestellt (prev - next), genauso wie bei den zusätzlichen Bildern auf der Artikeldetailseite.

Wenn Captions benötigt werden, können sie mit dem rel="" Attribut im Link angegeben werden:

<a href="images/bildgross.jpg" rel="here is my caption">Bild</a>


Lightbox auf jeder beliebigen Seite

Die Lightbox-Funktionalität kann leicht auf jede beliebige Seite in Zen-Cart ausgeweitet werden. Dazu muss die Datei jscript_zen_lightbox.php im Ordner ZUSATZ dieses Downloads in das entsprechende includes/modules/pages Verzeichnis gelegt werden.

Beispiele:
Lightbox-Effekt auf der Kontaktseite:
Die Datei muss nach includes/modules/pages/contact_us kopiert werden.
Lightbox-Effekt auf der Versandkostenseite:
Die Datei muss nach includes/modules/pages/shippinginfo kopiert werden

Update von älteren Zen Lightbox Versionen

Code, Dateistruktur und Dateinamen haben sich mit 1.5 stark verändert. Daher muss eine frühere Lightbox Version erst komplett entfernt werden:

Wenn von Version 1.4 auf 1.5.1 upgedated wird, unbedingt erst das Uninstall-Script ausführen, das bei 1.4 dabei war (Admin > Tools > SQL Patches installieren), um die alten Datenbankeinträge zu entfernen.

  1. Öffne den alten Download Deiner früheren Lightbox-Version und schau nach, welche Dateien enthalten waren. Lösche dann all diese Dateien vom Server.

  2. Folge dann der Anleitung für eine Installation wie weiter oben beschrieben.
