Scrolling Sideboxes 2.1 für Zen-Cart 1.3.9/1.3.8 deutsch

Übersicht | Installation

Installation


Voraussetzungen

Dieses Modul ist nur für Zen-Cart 1.3.9 oder 1.3.8 deutsch geeignet


Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

================= WICHTIG =====================
Vor dem Einbau dieser Änderungen:
BACKUP von Shop und Datenbank machen!
Keine Haftung, Verwendung auf eigene Gefahr!
BACKUP gemacht? Ok, dann weiterlesen...
===============================================


1.
Im Ordner NEUEDATEIEN dieses Downloads den Ordner js samt Inhalt ins Zen-Cart Verzeichnis hochladen. Dabei werden keine bestehenden Dateien überschrieben.

2.
Im Ordner GEAENDERTE DATEIEN dieses Downloads im Ordner zencart erst alle Ordner namens DEINTEMPLATE auf das im Shop verwendete Template umbenennen (z.B. classic, viennablood,...)
Dann in der vorgegebenen Struktur ins Zen-Cart Verzeichnis hochladen.
Wenn Sie den Effekt nur für bestimmte Sideboxen nutzen wollen (z.B. nur whats new) nur die entsprechenden Dateien hochladen. Dabei werden die bestehenden Dateien der Sideboxen überschrieben, aber dafür haben Sie ja vorher eine Sicherung gemacht, nicht wahr?

3.
Fertig. Mehr ist es nicht.
Die entsprechenden Sideboxen wie gewohnt über Admin > Tools > Boxlayout einschalten und positionieren.
