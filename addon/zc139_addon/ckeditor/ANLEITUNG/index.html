Sinn und Zweck dieses Moduls

Dieses Modul integriert den CKEditor als Alternative zu HTML Area in Zen-Cart.
Verwendet wird die Version 3.6.2 von http://www.ckeditor.com (erschienen September 2011).

Installation

Voraussetzungen


Dieses Modul ist nur für die deutsche Zen-Cart 1.3.9 Version geeignet


Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

WICHTIG
Vor dem Einbau dieser Änderungen:
BACKUP von Shop und Datenbank machen!
Keine Haftung, Verwendung auf eigene Gefahr!
BACKUP gemacht? Ok, dann weiterlesen...

Die Installation erfolgt in folgenden Schritten. Halten Sie diesen Ablauf ein!

1.
Im Ordner NEUE DATEIEN den Ordner DEINADMIN auf den Namen Ihres admin Verzeichnisses umbenennen. Dann in der vorgebenenen Struktur ins Zen-Cart Verzeichnis hochladen. Dabei werden keine bestehenden Dateien überschrieben.

2.
Im Ordner GEAENDERTE DATEIEN den Ordner DEINADMIN auf den Namen Ihres admin Verzeichnisses umbenennen. Beim Hochladen werden bestehende Dateien überschrieben. Es ist zwar sehr unwahrscheinlich, dass Sie diese Dateien schon einmal geändert haben, vergleichen Sie aber vor dem Hochladen zur Sicherheit Ihre bestehenden Dateien und nehmen Sie die Änderungen manuell vor.

3.
In der Shop-Administration können Sie nun auf den geeigneten Seiten (z.B. Artikelbearbeitung) rechts oben statt HTML Area den CKEditor auswählen.
Um den CKEditor als Standard zu setzen: Konfiguration > Mein Shop > HTML Editor
Möglicherweise müssen Sie sich nach dieser Änderung abmelden und den Browsercache leeren, damit der CKEditor sofort überall als Standard gesetzt ist

4.
Optional
Sie können den Ordner, in dem sich der Editor befindet (Standarmäßig editors) umbenennen. Das ist aus Sicherheitsgründen empfohlen, falls sie Zusatzplugins für den CKEditor verwenden.
Wenn Sie den Ordnernamen editors umbenennen, dann müssen Sie ihn auch in DEINADMIN/includes/configure.php entsprechend anpassen:
define('DIR_WS_EDITORS', 'neuername');

5.
Wenn Sie später einmal auf eine neuere Version des CKEditors aktualisieren wollen, einfach das neue Paket von www.ckeditor.com herunterladen und die Dateien in editors/ckeditor überschreiben
