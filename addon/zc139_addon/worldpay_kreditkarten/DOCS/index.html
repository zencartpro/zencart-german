WorldPay Kreditkarten Gateway 2.1 für Zen-Cart 1.3.9/1.3.8 deutsch

Übersicht | Installation

Sinn und Zweck dieses Moduls


Mit diesem Modul kann Kreditkartenzahlung via RBS WorldPay angeboten werden.
Die Einstellungsmöglichkeiten sind im deutschen Admin auf deutsch und im englischen Admin auf englisch.
Alle Einstellungsmöglichkeiten sind dort erklärt.
Nach der Installation ist in der Zen-Cart Administration unter Kunden der Menüpunkt WorldPay Zahlungen vorhanden, in dem alle per WorldPay getätigten Zahlungen sowie Rückmeldungen von WorldPay ersichtlich sind.

Dieses Modul ist nur für eine deutsche Zen-Cart 1.3.9 oder 1.3.8 Version geeignet

Es wird empfohlen, das Modul erst einmal in einem Testshop zu installieren und dort auszuprobieren.

Voraussetzung ist eine Anmeldung zu Business Gateway Plus bei RBS WorldPay.
Detaillierte Informationen dazu auf: http://www.rbsworldpay.com/deutsch/ecom/index.php?page=service&sub=worlddirect
Sobald ein Vertrag mit RBS WorldPay abgeschlossen wurde, steht bei WorldPay ein Testsystem für Test-Transaktionen zur Verfügung, das auch von diesem Modul unterstützt wird. Es können also erst alle Einstellungen im Testsystem vorgenommen, im Modul darauf angepasst und Testbestellungen durchgeführt werden. Später kann dann auf Live-Transaktion umgestellt werden.
