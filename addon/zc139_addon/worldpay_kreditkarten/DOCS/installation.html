WorldPay Kreditkarten Gateway 2.1 für Zen-Cart 1.3.9/1.3.8 deutsch

Übersicht | Installation

Installation


Voraussetzungen

  • Dieses Modul ist nur für Zen-Cart 1.3.9 oder 1.3.8 deutsch geeignet
  • Vertrag mit RBS WorldPay (Business Gateway Plus)

Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie vor der Installation unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

================= WICHTIG =====================
Vor dem Einbau dieser Änderungen:
BACKUP von Shop und Datenbank machen!
Keine Haftung, Verwendung auf eigene Gefahr!
BACKUP gemacht? Ok, dann weiterlesen...
===============================================


1.
Im Order NEUE DATEIEN im Unterordner zencart den Ordner DEINADMINVERZEICHNIS auf den Namen Ihres admin Verzeichnisses umbenennen.
Ihr Adminverzeichnis haben Sie ja umbenannt oder?
Falls nicht, nehmen Sie den Einbau dieses Moduls zum Anlass, das schleunigst zu tun. Eine Anleitung und weitere Informationen dazu finden Sie hier:
http://www.zen-cart.at/zcvb/forum/vbglossar.php?do=showentry&id=6

2.
Dann die Ordner/Dateien aus dem Unterordner zencart in der vorgegebenen Struktur in Ihr Zen-Cart Verzeichnis hochladen. Dabei werden keine bestehenden Dateien überschrieben.

3.
In der Zen-Cart Administration unter Module > Zahlungsarten das neu erschienene WorldPay Modul markieren. Auf "Installieren" und danach die nötigen Einstellungen vornehmen.

4.
Tip: Das Modul ist nach der Installation auf Testumgebung geschaltet. Richten Sie erst Ihre WorldPay Testumgebung ein und vergeben dann in der Zen-Cart Adminsitration die zur Testumgebung passenden Einstellungen. Ein großer Vorteil von WorldPay ist die Testumgebung, nutzen Sie sie solange, bis sie wirklich mit allen Einstellungen zufrieden sind!

5.
OPTIONAL
Dieses Modul liefert eine Sidebox mit, die die akzeptierten Kreditkarten anzeigt und auch das von WorldPay vorgeschriebene Worldpay Logo. Die Logos werden direkt von der WorldPay Seite via https geholt. Um diese Sidebox anzupassen, die includes/templates/template_default/sideboxes/tpl_wp_cc_accept.php bearbeiten. Die Sidebox kann wie gewohnt via Zen-Cart Administration (Tolls > Boxlayout) eingeschaltet und positioniert werden.
