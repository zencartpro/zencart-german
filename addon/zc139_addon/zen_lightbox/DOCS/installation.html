Installation


Voraussetzungen

Dieses Modul ist nur für Zen-Cart 1.3.9h deutsch oder 1.3.8a deutsch geeignet.

Für den Einbau dieses Moduls sollten Sie über folgende Werkzeuge verfügen:

1) Ein guter Texteditor.
Damit ist nicht das in Windows enthaltene Notepad oder Microsoft Word gemeint.
Sie benötigen einen Texteditor, der utf-8 versteht und auch im Format utf-8 ohne BOM abspeichern kann.
Meine Empfehlung: UltraEdit (kostenlose 30 Tage Testversion verfügbar)
Ebenfalls gut geeignet ist der kostenlose Texteditor Notepad++

2) Ein Tool zum Vergleichen von Dateien
Bei der Installation dieses Moduls ist es erforderlich, dass Sie den Inhalt einiger Ihrer bestehenden Zen-Cart Dateien mit dem Inhalt der neuen Moduldateien vergleichen und die Änderungen zusammenführen ("mergen").
Meine Empfehlung: BeyondCompare (kostenlose 30 Tage Testversion verfügbar)
Ebenfalls gut geeignet ist das kostenlose Programm WinMerge


Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

Neuinstallation


1.
Die Datei install.sql im Ordner SQL mit einem Texteditor öffen und den Inhalt kopieren

2.
Zen-Cart Administration öffnen und unter Tools > SQL Patches installieren den Inhalt ins Feld hineinkopieren und absenden.
Dadurch werden die Konfigurationsmöglichkeiten in der Datenbank angelegt.

3.
Neue Dateien des Moduls einspielen:
Im Ordner NEUE DATEIEN dieses Downloads im Ordner zencart alle Ordner namens DEINTEMPLATE auf den Namen des im Shop aktiven Templates umbenennen (z.B. classic, viennabright, etc.)
Dann die Ordner/Dateien in der vorgegebenen Struktur ins Zen-Cart Verzeichnis hochladen.
Dadurch werden keine bestehenden Dateien überschrieben

4.
Geänderte Dateien des Moduls einspielen.
Im Ordner GEAENDERTE DATEIEN dieses Downloads befinden sich im Ordner zencart die für dieses Modul notwendigen geänderten Originaldateien.
Es wird dringend empfohlen, die bestehenden Orginaldateien nicht einfach zu überschreiben!
Machen Sie das nur, wenn Sie Zen-Cart gerade neu installiert haben und noch keine Änderungen an Originaldateien vorgenommen haben.
Stattdessen vergleichen Sie Ihre entsprechenden Originaldateien mit einem geeigneten Programm wie z.B. WinMerge oder Beyond Compare und nehmen die Änderungen manuell vor.
Folgende Dateien müssen geändert werden (DEINTEMPLATE steht dabei für das im Shop aktive Template):
includes/modules/DEINTEMPLATE/additional_images.php
includes/templates/DEINTEMPLATE/templates/tpl_modules_main_product_image.php
Hinweis:
Falls Sie bereits das Modul Image Handler 2.3 im Shop aktiv haben, müssen Sie die additional_images.php nicht hochladen, da sie mit der aus Image Handler 2.3 identisch ist!

5.
In der Zen-Cart Administration ist nun unter Konfiguration ein zusätzlicher Menüpunkt namens Zen Lightbox vorhanden, wo Sie etliche Einstellungen zum Verhalten der Lightbox vornehmen können.

Erweiterte Nutzungsmöglichkeiten

EZ-Pages

Standardmäßig ist der Lightbox Effekt für alle Links zu Bildern innerhalb von EZ-Pages bereits aktiv. In der Lightbox-Konfiguration kann das ausgeschaltet werden.
Wenn Sie also ein Bild mit Link zum größeren Bild so in eine EZ-Page einbauen:

<a href="images/bildgross.jpg"><img src="images/bildklein.jpg"></a>

Dann wird das Bild beim Anclicken in einer Lightbox vergrößert.

Ebenso geht das Bild in einer Lightbox auf, wenn nur ein Link zum Bild gesetzt wird und der Link angecklickt wird:

<a href="images/bildgross.jpg">Bild</a>
Werden innerhalb eines übergeordneten Elements mehrere Links zu Bildern gesetzt, werden all diese Bilder in der Galerieansicht dargestellt (prev - next), genauso wie bei den zusätzlichen Bildern auf der Artikeldetailseite.

Wenn Captions benötigt werden, können sie mit dem rel="" Attribut im Link angegeben werden:

<a href="images/bildgross.jpg" rel="here is my caption">Bild</a>


Lightbox auf jeder beliebigen Seite

Die Lightbox-Funktionalität kann leicht auf jede beliebige Seite in Zen-Cart ausgeweitet werden. Dazu muss die Datei jscript_zen_lightbox.php im Ordner ZUSATZ dieses Downloads in das entsprechende includes/modules/pages Verzeichnis gelegt werden.

Beispiele:
Lightbox-Effekt auf der Kontaktseite:
Die Datei muss nach includes/modules/pages/contact_us kopiert werden.
Lightbox-Effekt auf der Versandkostenseite:
Die Datei muss nach includes/modules/pages/shippinginfo kopiert werden

Zen Lightbox 1.6.2 vollständig deinstallieren

1.
Die Datei uninstall.sql im Ordner SQL mit einem Texteditor öffen und den Inhalt kopieren

2.
Zen-Cart Administration öffnen und unter Tools > SQL Patches installieren den Inhalt ins Feld hineinkopieren und absenden.

3.
Alle von diesem Modul geänderten 1.3.9 Originaldateien wieder einspielen
Alle von diesem Modul neu angelegten Dateien vom Server löschen.
