pdf Invoice 3 for Zen-Cart 1.3.8

Overview | Installation | Configuration | FAQ

What does this module do?

An invoice in pdf format can be created in the Admin-Orders overview area.
This invoice can be generated automatically when the order is placed or the order status is changed.
The customer can download a copy from their order history.

Some features:

  • The invoice template can be easily changed using a separate pdf as a template
  • A wide range of fonts is included - more can be added
  • The pdf invoice can be automatically attached to the initial order confirmation mail if required or sent on an order status change
  • Offers the option of automatically including additional documents about terms & conditions or order cancellation
  • The pdf invoice can be downloaded by the customer from their order history under 'My account'
  • Includes utf-8 support
  • Automated installation - no manual SQL import required
  • and much more...
Example: http://demo.zen-cart.at/
