Übersicht über das PayPal (IPN) Instant Payment Notification Modul

  • Vollständig integriert in den Zen Cart™ Zahlungsvorgang.
  • PayPal's (IPN) Instant Payment Notification aktualisiert automatisch Ihre Shop Datenbank, egal ob ihr Kunde nach der erfolgreichen Bezahlung zurück auf Ihre Seite kommt oder nicht.
  • Alle PayPal IPN Informationen sind direkt in der Bestellübersicht im Adminbereich verfügbar.
  • Vollständige Unterstützung für die in Zen Cart™ verfügbaren Features Aktionskupons, Geschenkgutscheine, downloadbare Produkte und Versandoptionen.
  • Unterstützt PayPal Instant Payments, E-Schecks, Kreditkartenzahlungen und zahlreiche Währungen.
  • Bestellstatus Änderungen abhängig von der Art der Bezahlung und/oder Rückzahlung/Ablehnung der Zahlung/Abbruch der Zahlung.

Voraussetzungen

  • Eine komplette funktionierende Installation von Zen Cart™
  • Paypal Premium oder Händler Account.

    Einige Account Einstellungen müssen Sie direkt in Ihrem Paypal Account vornehnem
    Für weitere Informatioenen lesen die die FAQ unten.

PayPal IPN Modul Installation

Installieren des Moduls

  • Öffnen Sie den Zen Cart™ Adminbereich.
  • Klicken Sie auf Module in der Menüzeile.
  • Wählen Sie Zahlungsarten vom Dropdown Menu.
  • Klicken Sie auf das Icon PayPal IPN.
  • Klicken Sie auf den Install Button.
Gratulation! Sie haben soeben das PayPAl IPN Modul installiert.

Update des Moduls (oder Aktivierung der Änderungen in einer neuen Version)

Dieses ist nur dann von Interesse, wenn Sie eine bestehende Zen Cart™ Installation aktualisiert haben:
  • Öffnen Sie den Zen Cart™ Adminbereich.
  • Klicken Sie auf Module in der Menüzeile.
  • Wählen Sie Zahlungsarten vom Dropdown Menu.
  • Klicken Sie auf das Icon PayPal IPN.
  • Klicken Sie auf den Bearbeiten Button .
  • Notieren Sie sich alle Einstellungen.
  • Klicken Sie auf OK.
  • Klicken Sie auf den Entfernen Button. Dieses entfernt das Modul und alle alten Einstellungen.
  • Klicken Sie auf den Install Button, um die neue Version dieses Moduls zu installieren.
  • Geben Sie die eben notierten Einstellungen erneut ein.
Gratulation! Sie haben soeben das PayPAl IPN Modul installiert.

PayPal IPN Module Konfiguration

Konfiguration des Moduls

  • Enable PayPal Module - Wollen Sie PayPal Zahlungen akzeptieren?
    • True
      Wählen Sie True falls es nicht standardmäßig ausgewählt wird
    • False
      Falls Sie False auswählen, wird das Modul deaktiviert und steht Ihren Kunden nicht mehr als Zahlungsmöglichkeit zur Verfügung. Es wird in der Bestellabwicklung nicht mehr angezeigt.
  • Business/Händler ID - Ihre primäre E-Mail Adresse Ihres PayPal Accounts
    Es ist wichtig, das Sie die primäre E-Mail Adresse Ihres PayPay Accounts verwenden.
    HINWEIS: Die angegebene E-Mail Adresse muss EXAKT der angegebenen primären E-Mail Adresse in ihren PayPal Account Einstellungen entsprechen. Achtung, es wird Groß- / Kleinschreibung unterschieden.


  • Transaction Currency - Wählen Sie die Währung/Währungen, welche Sie akzeptieren wollen.
    • Selected Currency (Ihr Kunde wählt die Währung aus)
      Wählen Sie diese Option aus, um verschiedene Währungen zu akzeptieren.

    • Only USD
      Wählen Sie diese Option aus, wenn Sie Zahlungen nur in US Dollar akzeptieren wollen.

    • Only CAD
      Wählen Sie diese Option aus, wenn Sie Zahlungen nur in Kanadischen Dollar akzeptieren wollen.

    • Only EUR
      Wählen Sie diese Option aus, wenn Sie Zahlungen nur in Euro akzeptieren wollen.

    • Only GBP
      Wählen Sie diese Option aus, wenn Sie Zahlungen nur in Pfund akzeptieren wollen.

    • Only JPY
      Wählen Sie diese Option aus, wenn Sie Zahlungen nur in Yen akzeptieren wollen.

    • Only AUD
      Wählen Sie diese Option aus, wenn Sie Zahlungen nur in Australischen Dollar akzeptieren wollen.

  • Payment Zone - Wenn Sie eine Zone auswählen, dann können nur Kunden innerhalb dieser Zone per PayPal bezahlen. Alle anderen Kunden bekommen PayPal während der Bestellabwicklung nicht zur Auswahl angeboten.

  • Set Pending Notification Status - Setzt den Status von Bestellungen mit diesem Zahlungsmodul auf diesen Wert, solange die Zahlung noch nicht abgeschlossen ist.('Wartet' empfohlen)

  • Set Order Status - Setzt den Status von Bestellungen mit diesem Zahlungsmodul auf diesen Wert, sobald die Zahlung erfolgreich abgeschlossen wurde. ('In Bearbeitung' empfohlen)
    (Dies ist der Status, welcher automatisch gesetzt wird, wenn Paypal die erfolgreiche via IPN bestätigt.)

  • Set Refund Order Status - Setzt den Status von Bestellungen mit diesem Zahlungsmodul auf diesen Wert, sofern der Betrag ganz oder teilweise zurückgezahlt wurde. ('Wartet' empfohlen)

  • Sort order of display - Sortierung der Anzeige ihrer Zahlungsmodule während des Bestellvorgangs. Je kleiner der Wert, desto weiter oben steht die jeweilige Zahlungsart.

  • Continue Button Text - Wenn der Kunde auf der PayPal Website zur Bezahlung ist, wird nach der erfolgreichen Bezahlung ein "Continue" Button angezeigt. Dieser bringt den Kunden zurück auf Ihre Shopwebsite. Sollten Sie einen anderen Text als "Continue" wünschen, können Sie diesen hier eintragen. Ansonsten lassen Sie das Feld bitte leer (Standard = leer).

  • Page Style - Sie können das Erscheinungsbild der PayPal Website anpassen, welches der Kunde währen der Bezahlung zu Gesicht bekommt. Wenn Sie diese Website verändern, dann können Sie dieser einen eigenen Namen geben. Der Standard Name ist "paypal", und "paypal" ist der Name, der benutzt wird, wenn Sie keinen anderen angeben.
    Wenn Sie eine eigenes PayPal Seitenlayout (aka Template) erstellt habe, dann können Sie hier Ihren Layout Namen eingeben.
    Wenn Sie hier "paypal" eingeben, verwenden Sie das Standardlayout der PayPal Seite.

  • Mode for PayPal web services - Wählen Sie die URL für PayPal Live oder Test Service
    • www.paypal.com/cgi-bin/webscr
            Wählen Sie diese URL für Live Service
    • (evtl. müssen die www.paypal.com/us/cgi-bin/webscr benutzen)

  • Debug Mode - Sie können alle PayPal IPN Aktivitäten mitloggen lassen. Diese macht nur Sinn, wenn die auf Fehlersuche mit einem PayPal Mitarbeiter sind. Wenn Sie "email" auswählen, erhalten Sie das Logfile an die E-Mail Adresse, welche unter Shopbesitzer/Store Owner angegeben ist. Sollten Sie die Logfiles nicht erhalten, schauen die ggfs. in Ihren Junkmail Ordner. Die Logs werden im Ordner /includes/modules/paypal/logs gespeichert, bitte sorgen Sie für die entsprechenden Lese-/Schreibrechte.

Frequently Asked Questions (FAQ)

Muss ich IPN in meinem PayPal Account aktivieren? Welche URL sollte ich angeben, um IPN nutzen zu können?

Ja, Sie müssen es aktivieren.
1. Loggen Sie sich in Ihren PayPal Account ein und gehen Sie auf Profil
2. Klicken Sie auf "Instant Payment Notification Einstellungen"
3. Klicken Sie auf Bearbeiten
Die URL sollte auf http://www.meinedomain.de/ipn_main_handler.php verweisen
oder wenn Sie Zen Cart™ in einem Unterverzeichnis haben:
http://www.meinedomain.de/unterverzeichnis/ipn_main_handler.php


Muss ich die Auto-Return Funktion in meinem PayPal Account aktivieren?

Ja. Das Aktivieren der Auto-Return Funktion bringt Ihren Kunden automatisch 5-10 Sekunden nach dem erfolgreichen Abschluss der Zahlung zurück in Ihren Shop.

1. Loggen Sie Sich in Ihren PayPal Account ein und gehen Sie auf Profil
2. Klicken Sie auf "Website Payments Einstellungen"
3. Aktivieren Sie Auto-Return
4. Geben Sie eine Rückkehr URL ein (Return URL):
Für nonSSL Shops:
http://www.meineseite.de/MEIN_SHOP/index.php?main_page=checkout_process
or
Für Shops mit eigenem SSL Zertifikat:
https://www.meineseite.de/MEIN_SHOP/index.php?main_page=checkout_process
or
https://mysite.secureserver.net/MEIN_SHOP/index.php?main_page=checkout_process

Sollte ich "PayPal Account Optional" aktivieren?
Das können Sie ganz nach Ihren Wünschen handhaben.
