EIN SPEZIELLER HINWEIS ZU DEN .htaccess DATEIEN

In einigen Verzeichnissen sind .htaccess Dateien enthalten, die eine Liste von zugelassenen Dateitypen, auf die zugegriffen werden darf, enthalten. Alle Zugriffsversuche auf nicht zugelassene Dateiendungen werden blockiert, um Ihren Shop vor einem unerlaubten Zugriff zu schützen.

Ein Nebeneffekt davon ist, wenn Sie Dateitypen in Ihrem Shop verwenden, die nicht in der .htaccess Datei stehen, haben Ihre Kunden keinen Zugriff darauf.

Deshalb sollten Sie, wenn Sie solche gesperrten Dateitypen für Artikelbilder verwenden wollen, die .htaccess Datei in Ihrem /images/ Verzeichnis entsprechend anpassen.

Das Gleiche gilt für gesperrte Dateitypen im Media Verzeichnis, um diese zuzulassen ändern Sie bitte die .htaccess Datei im Verzeichnis /media/.

Für gesperrte Download Dateitypen ändern Sie bitte die .htaccess Dateien in den Verzeichnissen /pub/ und /download/.
