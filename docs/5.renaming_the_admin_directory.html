Sicherheit Ihres Adminverzeichnisses

Um was geht es hier?

Wir empfehlen Ihnen, Ihr Admin Verzeichnis direkt nach der Installation umzubenennen.
Das erschwert Hackern das Finden Ihres Adminbereichs erheblich.

Wie benenne ich mein Admin Verzeichnis um?

1- Öffnen Sie Ihre Admin Konfigurationsdatei: admin/includes/configure.php

Ändern Sie in dieser Datei alle Einträge von /admin/ in den von Ihnen ausgewählten Namen:

Ändern Sie in diesem Bereich:

// NOTE: be sure to leave the trailing '/' at the end of these lines if you make changes!
// * DIR_WS_* = Webserver directories (virtual/URL)
// these paths are relative to top of your webspace ... (ie: under the public_html or httpdocs folder)
define('DIR_WS_ADMIN', '/admin/');
define('DIR_WS_CATALOG', '/');
define('DIR_WS_HTTPS_ADMIN', '/admin/');
define('DIR_WS_HTTPS_CATALOG', '/');

und in diesem Bereich:

// * DIR_FS_* = Filesystem directories (local/physical)
//the following path is a COMPLETE path to your Zen Cart files. eg: /var/www/vhost/accountname/public_html/store/
define('DIR_FS_ADMIN', '/home/mystore.com/www/public/admin/');
define('DIR_FS_CATALOG', '/home/mystore.com/www/public/');

2- Nennen Sie Ihr Adminverzeichnis in den gewünschten Namen um.

Nutzen Sie dazu Ihr FTP Programm und suchen Sie Ihr /admin/ Verzeichnis. Benennen Sie das Verzeichnis um und verwenden Sie dabei den Namen, den Sie in Schritt 1 in der Datei admin/includes/configure.php angegeben haben.
