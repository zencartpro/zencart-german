Image Handler 3.0 für Zen-Cart 1.3.9 und 1.3.8 deutsch

Übersicht | Installation | Konfiguration

Installation


Voraussetzungen

Dieses Modul ist nur für Zen-Cart 1.3.9 oder 1.3.8 deutsch geeignet

Für den Einbau dieses Moduls sollten Sie über folgende Werkzeuge verfügen:

1) Ein guter Texteditor.
Damit ist nicht das in Windows enthaltene Notepad oder Microsoft Word gemeint.
Sie benötigen einen Texteditor, der utf-8 versteht und auch im Format utf-8 ohne BOM abspeichern kann.
Meine Empfehlung: UltraEdit (kostenlose 30 Tage Testversion verfügbar)
Ebenfalls gut geeignet ist der kostenlose Texteditor Notepad++

2) Ein Tool zum Vergleichen von Dateien
Bei der Installation dieses Moduls ist es erforderlich, dass Sie den Inhalt einiger Ihrer bestehenden Zen-Cart Dateien mit dem Inhalt der neuen Moduldateien vergleichen und die Änderungen zusammenführen ("mergen").
Meine Empfehlung: BeyondCompare (kostenlose 30 Tage Testversion verfügbar)
Ebenfalls gut geeignet ist das kostenlose Programm WinMerge


Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

================= WICHTIG =====================
Vor dem Einbau dieser Änderungen:
BACKUP von Shop und Datenbank machen!
Keine Haftung, Verwendung auf eigene Gefahr!
BACKUP gemacht? Ok, dann weiterlesen...
===============================================


1.
Im Order NEUEDATEIEN im Unterordner zencart den Ordner DEINADMINVERZEICHNIS auf den Namen Ihres admin Verzeichnisses umbenennen.
Alle Ordner namens DEINTEMPLATE auf das im Shop aktive Template umbenennen (z.B. classic, viennablood,...)
Dann die Ordner/Dateien in der vorgegebenen Struktur in Ihr Zen-Cart Verzeichnis hochladen. Dabei werden keine bestehenden Dateien überschrieben.

2.
Im Ordner GEAENDERTE DATEIEN im Unterordner zencart alle Ordner namens DEINTEMPLATE in den Namen des aktuell verwendeten Templates umbenennen (z.B. classic, viennablood,...)
Den Ordner DEINADMINVERZEICHNIS auf den Namen Ihres admin Verzeichnisses umbenennen.
Achtung:
Beim Hochladen dieser Dateien werden bestehende Dateien überschrieben!
Daher:
Falls bereits einmal Änderungen an diesen Dateien in der Zen-Cart Installation gemacht wurden, mit WinMerge oder BeyondCompare die Dateien vergleichen und nur die Änderungen in die bestehenden Dateien übernehmen.
Wurden noch keine Änderungen gemacht (Zen-Cart wurde z.B. gerade erst frisch installiert), die Ordner/Dateien aus dem Ordner GEAENDERTE DATEIEN einfach in der vorgegebenen Struktur in Ihr Zen-Cart Verzeichnis hochladen.

3.
Der Ordner bmz_cache, den Sie gerade in Ihr Shopverzeichnis hochgeladen haben benötigt Schreibrechte. In diesem Ordner werden später von Image Handler dynamisch generierte Bilddateien abgelegt. Geben Sie diesem Ordner daher Schreibrechte (chmod 777)

4.
In der Zen-Cart Administration ist nun unter Tools der Menüpunkt Image Handler 3.0 vorhanden
Rufen Sie dieses Menü auf und clicken Sie auf den Link "Image Handler installieren".
Dadurch werden die nötigen Datenbankeinträge vorgenommen und Sie sollten danach angezeigt bekommen:
Version 3.0

5.
Wechseln Sie nun ins Menü Konfiguration > Bilder
Hier sind durch die Image Handler Installation neue Konfigurationsmöglichkeiten hinzugekommen.
Stellen Sie hier Ihre gewünschten Optionen ein, Hinweise dazu finden Sie in dieser Anleitung unter Konfiguration

 

Update von einer älteren Image Handler Version

Wichtig:
Falls Sie bereits eine ältere Image Handler Version im Einsatz haben, entfernen Sie diese vorher komplett und folgen erst dann den Schritten unter Neuinstallation.

Entfernen einer älteren Image Handler Version
Im Ordner SQL dieses Downloads die uninstall.sql mit einem Texteditor öffnen und den Inhalt kopieren.
Dann in der Zen-Cart Administration unter Tools > SQL Patches installieren ins Eingabefeld kopieren und Absenden
Dadurch werden alle Datenbankeinträge auch sehr alter Image Handler Versionen vollständig entfernt.

Image Handler 3.0 vollständig deinstallieren

Zum Entfernen von Image Handler 3.0 rufen Sie auf:
Admin > Tools > Image Handler 3.0
Gehen Sie dort auf Admin und dann auf "Image Handler aus der Datenbank entfernen"
Löschen Sie dann alle von der alten Image Handler Installation neu angelegten Dateien vom Server und spielen Sie die Originaldateien ein, die von Image Handler überschrieben wurden.
