Google Analytics/Conversion Tracking 1.3.2 für Zen-Cart 1.3.x deutsch

Übersicht | Installation

Sinn und Zweck dieses Moduls


Mit diesem Modul kann der Google Analytics Code in den Shop integriert werden.
Falls auch Google Adwords verwendet wird, kann optional das Conversion Tracking aktiviert werden.
Die Eingabe der Codes erfolgt via Zen-Cart Admin, es ist dadurch nicht mehr nötig, die Google Tracking Javascripts in Header- Footer-, oder Checkout-Success-Templates einzubauen.
Sollte Google-Analytics bereits im Shop integriert sein, müssen die Javascripts vor Einbau des Moduls natürlich aus den jeweiligen Templates entfernt werden.
Die Version 1.3.2 bringt eine Unterstützung für den neuen asynchronen Google Analytics Code und ermöglicht zusätzliche benutzerdefinierte Trackingeinstellungen.

Geeignet für die deutsche Zen-Cart 1.3.9 oder 1.3.8 Version
