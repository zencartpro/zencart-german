Installation


Voraussetzungen

Dieses Modul ist nur für die deutsche Zen-Cart Version geeignet


Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

================= WICHTIG =====================
Vor dem Einbau dieser Änderungen:
BACKUP von Shop und Datenbank machen!
Keine Haftung, Verwendung auf eigene Gefahr!
BACKUP gemacht? Ok, dann weiterlesen...
===============================================


1.
Die Datei install.sql im Ordner SQL mit einem Texteditor öffen und den Inhalt kopieren

2.
Zen-Cart Administration öffnen und unter Tools > SQL Patches installieren den Inhalt ins Feld hineinkopieren und absenden.
Dadurch werden die nötigen Datenbankeinträge vorgenommen und in der Zen-Cart Administration ist unter Konfiguration der neue Menüpunkt Google Analytics Einstellungenverfügbar.

3.
Neue Dateien des Moduls einspielen:
Im Ordner NEUE DATEIEN dieses Downloads im Ordner zencart den Ordner DEINTEMPLATE auf den Namen des im Shop aktiven Templates umbenenennen (z.B. classic, viennablood, ...)
Den Ordner DEINADMINVERZEICHNIS auf den Namen Ihres admin Verzeichnisses umbenennen.
Dann in der vorgegebenen Struktur ins Zen-Cart Verzeichnis hochladen.
Dadurch werden keine bestehenden Dateien überschrieben

4.
In der Zen-Cart Administration unter Konfiguration > Google Analytics Einstellungen die gewünschten Einstellungen vornehmen.

5.
Folgende Datei aus der bestehenden Zen-Cart Installation vom Server herunterladen:
includes/templates/DEINTEMPLATE/common/tpl_main_page.php
Sollte im eigenen Templateverzeichnis keine common/tpl_main_page.php vorhanden sein, folgende Datei verwenden:
includes/templates/template_default/common/tpl_main_page.php
Datei mit einem Texteditor öffnen und ganz am Ende vor dem </body> folgenden Code einfügen:

<?php
require($template->get_template_dir('.php',DIR_WS_TEMPLATE, $current_page_base,'google_analytics') . '/google_analytics.php');
?>

Datei abspeichern und hochladen nach:
includes/templates/DEINTEMPLATE/common/tpl_main_page.php


Modul komplett deinstallieren

Um das Modul komplett zu entfernen:
Folgende Dateien wieder vom Server löschen:
DEINADMINVERZEICHNIS/includes/extra_datafiles/google_analytics_database_names.php
DEINADMINVERZEICHNIS/includes/functions/extra_functions/google_analytics_functions.php
Folgenden Ordner samt Inhalt löschen:
includes/templates/DEINTEMPLATE/google_analytics
In der includes/templates/DEINTEMPLATE/commeon/tpl_main_page.php den zuvor vor dem </body> Tag eingefügten Code entfernen und die Originaldatei wieder hochladen
Im Ordner SQL befindet sich eine uninstall.sql. Inhalt unter Admin > Tools > SQL Patches installieren ins Eingabefeld kopieren und absenden. Dadurch werden sämtliche Datenbankeinträge des Moduls gelöscht.
