Wie haben Sie von uns erfahren 2.0 für Zen-Cart 1.3.9 deutsch

Übersicht | Installation

Installation


Voraussetzungen

Dieses Modul ist nur für Zen-Cart 1.3.9 deutsch geeignet

Für den Einbau dieses Moduls sollten Sie über folgende Werkzeuge verfügen:

1) Ein guter Texteditor.
Damit ist nicht das in Windows enthaltene Notepad oder Microsoft Word gemeint.
Sie benötigen einen Texteditor, der utf-8 versteht und auch im Format utf-8 ohne BOM abspeichern kann.
Meine Empfehlung: UltraEdit (kostenlose 30 Tage Testversion verfügbar)
Ebenfalls gut geeignet ist der kostenlose Texteditor Notepad++

2) Ein Tool zum Vergleichen von Dateien
Bei der Installation dieses Moduls ist es erforderlich, dass Sie den Inhalt einiger Ihrer bestehenden Zen-Cart Dateien mit dem Inhalt der neuen Moduldateien vergleichen und die Änderungen zusammenführen ("mergen").
Meine Empfehlung: BeyondCompare (kostenlose 30 Tage Testversion verfügbar)
Ebenfalls gut geeignet ist das kostenlose Programm WinMerge


Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

================= WICHTIG =====================
Vor dem Einbau dieser Änderungen:
BACKUP von Shop und Datenbank machen!
Keine Haftung, Verwendung auf eigene Gefahr!
BACKUP gemacht? Ok, dann weiterlesen...
===============================================


1.
Die Datei install.sql im Ordner SQL mit einem Texteditor öffen und den Inhalt kopieren

2.
Zen-Cart Administration öffnen und unter Tools > SQL Patches installieren den Inhalt ins Feld hineinkopieren und absenden.
Dadurch wird ein zusätzliches Feld in der Tabelle customers_info angelegt und die Einstellungsmöglichkeiten in die Konfiguration geschrieben. Außerdem werden zwei neue Tabellen angelegt (sources und sources_other).

3.
Neue Dateien des Moduls einspielen:
Im Ordner NEUE DATEIEN dieses Downloads im Ordner zencart den Ordner DEINADMINVERZEICHNIS auf den Namen Ihres admin Verzeichnisses umbenennen
Dann die Ordner/Dateien in der vorgegebenen Struktur ins Zen-Cart Verzeichnis hochladen.
Dadurch werden keine bestehenden Dateien überschrieben

4.
Geänderte Dateien des Moduls einspielen.
Im Ordner GEAENDERTE DATEIEN dieses Downloads befinden sich im Ordner zencart die für dieses Modul notwendigen geänderten Originaldateien.
Es wird dringend empfohlen, die bestehenden Orginaldateien nicht einfach zu überschreiben!
Machen Sie das nur, wenn Sie Zen-Cart gerade neu installiert haben und noch keine Änderungen an Originaldateien vorgenommen haben.
Benennen Sie die Ordner DEINTEMPLATE auf das im Shop aktive Template um (z.B. classic, viennablood,...)
Benennen Sie den Ordner DEINADMINVERZEICHNIS aud den Namen Ihres admin Verzeichnisses um.
Dann vergleichen Sie Ihre entsprechenden Originaldateien mit einem geeigneten Programm wie z.B. WinMerge oder Beyond Compare und nehmen die Änderungen manuell vor.
Dann die geänderten Dateien in der vorgegebenen Struktur ins Zen-Cart Verzeichnis hochladen.

5.
In der Zen-Cart Administration ist nun unter Konfiguration > Mein Shop ein zusätzlicher Menüpunkt namens Zeige die Option "Andere" bei Wie haben Sie von uns erfahren? vorhanden.
Hier können Sie einstellen, ob sie neben den vorgegebenen Optionen auch das Freitextfeld bei Auswahl der Option "Andere" anzeigen wollen.
Unter Konfiguration > Kundendetails ist nun ein neuer Menüpunkt namens Wie haben Sie von uns erfahren als Pflichtfeld? vorhanden.
Hier stellen Sie ein, ob der Kunde bei der Registrierung nur weiterkommt, wenn er die Frage beantwortet hat.

6.
Bearbeiten Sie nun die mitgelieferten Optionen und passen Sie die Vorgabewerte auf Ihre Wünsche an.
Unter Extras > Wie haben Sie von uns erfahren? - Administration können Sie die voreingestellten Antworten bearbeiten, löschen oder neue hinzufügen

7.
Spielen Sie nun eine Registrierung mit Angabe einer Herkunftsart durch.
Unter Kunden > Kunden muss danach bei "Herkunfsart" die gegebene Antwort ersichtlich sein.
Einen generellen Überblick über alle gegebenen Antworten liefert Extras > Wie haben Sie von uns erfahren - Kundenantworten.
Doppelclick auf die Option "Andere" zeigt dann die Texte, die per Freitextfeld angegeben wurden.
Wenn Ihr Shop schon länger im Einsatz war, bevor Sie das Modul eingebaut haben, werden unter "Andere" auch die bisherigen Registrierungen mitgezählt, bei denen die Frage noch gar nicht gestellt wurde.
Bei Bestellungen ist nun unter "Bestellung bearbeiten" unter "Referral" ersichtlich ob und wenn ja welche Herkunft der Kunde angegeben hat.

Wie haben Sie von uns erfahren vollständig deinstallieren

1.
Die Datei uninstall.sql im Ordner SQL mit einem Texteditor öffen und den Inhalt kopieren

2.
Zen-Cart Administration öffnen und unter Tools > SQL Patches installieren den Inhalt ins Feld hineinkopieren und absenden.

3.
Alle von diesem Modul geänderten 1.3.9 Originaldateien wieder einspielen
Alle von diesem Modul neu angelegten Dateien vom Server löschen.
