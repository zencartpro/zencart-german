Sinn und Zweck dieses Moduls


Bei der Registrierung im Shop können Kunden angeben, wie sie vom Shop erfahren haben (Google, Empfehlung eines Bekannten, Twitter, Facebook, usw.)
Es erscheint ein Dropdownmenü mit den verschiedenen Optionen. Welche Optionen angeboten werden, kann in der Zen-Cart Administration eingestellt werden. Optional kann auch ein Freitextfeld angeboten werden.
Die Frage kann optional als Pflichtfeld definiert werden.
Die Antworten der Kunden sind dann in der Zen-Cart Administration ersichtlich.
