Facebook Like It 1.0 für Zen-Cart 1.3.9 deutsch

Übersicht | Installation

Installation


Voraussetzungen

Dieses Modul ist nur für die deutsche Zen-Cart 1.3.9 Version geeignet

Sie benötigen einen Facebook API Key.
Falls Sie noch keinen erstellt haben:
http://www.facebook.com/developers

Für den Einbau dieses Moduls sollten Sie über folgende Werkzeuge verfügen:

1) Ein guter Texteditor.
Damit ist nicht das in Windows enthaltene Notepad oder Microsoft Word gemeint.
Sie benötigen einen Texteditor, der utf-8 versteht und auch im Format utf-8 ohne BOM abspeichern kann.
Meine Empfehlung: UltraEdit (kostenlose 30 Tage Testversion verfügbar)
Ebenfalls gut geeignet ist der kostenlose Texteditor Notepad++

2) Ein Tool zum Vergleichen von Dateien
Bei der Installation dieses Moduls ist es erforderlich, dass Sie den Inhalt einiger Ihrer bestehenden Zen-Cart Dateien mit dem Inhalt der neuen Moduldateien vergleichen und die Änderungen zusammenführen ("mergen").
Meine Empfehlung: BeyondCompare (kostenlose 30 Tage Testversion verfügbar)
Ebenfalls gut geeignet ist das kostenlose Programm WinMerge


Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

================= WICHTIG =====================
Vor dem Einbau dieser Änderungen:
BACKUP von Shop und Datenbank machen!
Keine Haftung, Verwendung auf eigene Gefahr!
BACKUP gemacht? Ok, dann weiterlesen...
===============================================


1.
Im Order NEUEDATEIEN im Ordner zencart die Datei likeit_de.js mit einem Texteditor öffnen und Ihren Facebook API Key eintragen:

FB.init({appId: 'hierIhrenAPIKeyeintragen', status: true, cookie: true,
Sollte also danach etwa so aussehen:

FB.init({appId: 'ab65tcgfgdf234rfg897gftdhhjjh675rd3r', status: true, cookie: true,
Dasselbe in der likeit_en.js machen.

Abspeichern und dann die Dateien im Ordner zencart in der vorgebenenen Struktur ins Zen-Cart Verzeichnis hochladen.
Dabei werden keine bestehenden Dateien überschrieben.

2.
Im Ordner GEAENDERTE DATEIEN dieses Downloads befinden sich im Ordner zencart die für dieses Modul notwendigen geänderten Originaldateien.
Es wird dringend empfohlen, die bestehenden Orginaldateien nicht einfach zu überschreiben!
Machen Sie das nur, wenn Sie Zen-Cart gerade neu installiert haben und noch keine Änderungen an Originaldateien vorgenommen haben.
Stattdessen vergleichen Sie Ihre entsprechenden Originaldateien mit einem geeigneten Programm wie z.B. WinMerge oder Beyond Compare und nehmen die Änderungen manuell vor.
Folgende Dateien müssen geändert werden (DEINTEMPLATE steht dabei für das im Shop aktive Template):
includes/templates/DEINTEMPLATE/common/html_header.php
includes/templates/DEINTEMPLATE/common/tpl_main_page.php
Sollten diese beiden Dateien im eigenen Templateverzeichnis nicht vorhanden sein, verwenden Sie die entsprechenden aus includes/templates/template_default
Nach den Änderungen hochladen in includes/templates/DEINTEMPLATE/common

3.
Sprachdefinitionen ergänzen
Öffnen Sie Ihre bestehende includes/languages/german/DEINTEMPLATE/product_info.php
und tragen Sie dort folgende neue Definition ein: // facebook likeit
define('TEXT_FACEBOOKCONNECT','<script type="text/javascript" src="js/likeit_de.js"></script>');

Öffnen Sie Ihre bestehende includes/languages/english/DEINTEMPLATE/product_info.php
und tragen Sie dort folgende neue Definition ein: // facebook likeit
define('TEXT_FACEBOOKCONNECT','<script type="text/javascript" src="js/likeit_en.js"></script>');

4.
Öffnen Sie Ihre bestehende includes/templates/DEINTEMPLATE/tpl_product_info_display.php mit einem Texteditor und fügen Sie folgenden Code an die gewünschte Stelle ein, an der Sie den Facebook Like It Button haben wollen: <!--bof likeit-->
<?php if ($_SERVER['https'] != 'on') { ?>
<div id="fb-root"></div>
<?php echo TEXT_FACEBOOKCONNECT; ?>
<!-- FBML <fb:like href="<?php echo urlencode(zen_href_link(zen_get_info_page((int)$_GET['products_id']),'cPath='.(int)$_GET['cPath'].'&products_id=' . (int)$_GET['products_id'])); ?>"></fb:like> FBML -->
<?php } ?>
<!--eof likeit-->


Falls im Shop weitere Sprachen aktiv sind, die entsprechenden Javascripts erstellen und die entsprechenden Sprachdefinitionen auch bei den übrigen Sprachen nachtragen.

Modul komplett deinstallieren

Um das Modul komplett zu entfernen:
Alle oben beschriebenen Änderungen wieder rückgängig machen und die Orignaldateien wieder einspielen.
