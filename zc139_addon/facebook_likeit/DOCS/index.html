Facebook Like It 1.0 für Zen-Cart 1.3.9 deutsch

Übersicht | Installation

Sinn und Zweck dieses Moduls


Mit diesem Modul kann der Facebook Like It Button w3c-konform und passend zur jeweiligen Sprache auf den Artikeldetailseiten eingebunden werden.
Besucht der Kunde die deutsche Version wird die deutsche Variante angeboten, besucht er die englische, die englische Variante. Sollten weitere Sprachen aktiv sein, können diese leicht ergänzt werden.

Voraussetzungen:

  • ein Facebook Account
  • ein Facebook API Code
