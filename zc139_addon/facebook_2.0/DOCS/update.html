Update von Facebook 2.0 auf Facebook 2.1

 

Wenn Sie bereits Version 2.0 dieses Moduls verwenden und auf 2.1 updaten wollen, gehen Sie bitte folgendermaßen vor:

1)
Aktualisieren des CSS/Javascript Loaders
Version 2.1 enthält eine neue Version des CSS/Javascript Loaders.
Die Datenbankkonfigurationseinstellungen unter Konfiguration > CSS/JS Loader bleiben gleich.
Es haben sich allerdings einige Dateien geändert und neue sind hinzugekommen.

Laden Sie zuerst wie unter Neuinstallation beschrieben alle Dateien und Ordner aus dem Ordner NEUE DATEIEN im Ordner SCHRITT 1 - CSS JAVASCRIPT LOADER hoch (vorher den darin enthaltenen Ordner includes/DEINTEMPLATE auf den Namen des im Shop aktiven Templates umbenennen. Dabei werden bestehende alte Dateien des Moduls überschrieben, das ist ok.

Passen Sie nun folgende Datei aus Ihrer Zen-Cart Installation an:
includes/templates/DEINTEMPLATE/common/html_header.php
Diese enthält noch den alten Code zum Aufruf des CSS/Javascript Loaders.
Vergleichen Sie die Datei mit der aus dem Ordner GEAENDERTE DATEIEN im Ordner SCHRITT 1 - CSS JAVASCRIPT LOADER und ändern Sie Ihre Datei entsprechend ab.

Entfernen Sie folgende Dateien vom Server, sie werden nun nicht mehr benötigt:
includes/classes/browser.php
includes/classes/class.cj_loader.php
includes/auto_loaders/config.css_jscript_auto_loaders.php
includes/init_includes/init_css_jscript_auto_loaders.php

2)
Aktualisierung von Facebook Open Graph
Kopieren Sie folgenden Befehl unter Tools > SQL Patches installieren ins Eingabefeld und senden ihn ab:

ALTER TABLE `products` DROP `stored_url`;
DELETE FROM configuration WHERE configuration_key = 'FACEBOOK_OPEN_GRAPH_CPATH';
DELETE FROM configuration WHERE configuration_key = 'FACEBOOK_OPEN_GRAPH_LANGUAGE';
DELETE FROM configuration WHERE configuration_key = 'FACEBOOK_OPEN_GRAPH_CANONICAL';
DELETE FROM configuration WHERE configuration_key = 'FACEBOOK_OPEN_GRAPH_STORED_URL';
DELETE FROM configuration_language WHERE configuration_key = 'FACEBOOK_OPEN_GRAPH_CPATH';
DELETE FROM configuration_language WHERE configuration_key = 'FACEBOOK_OPEN_GRAPH_LANGUAGE';
DELETE FROM configuration_language WHERE configuration_key = 'FACEBOOK_OPEN_GRAPH_CANONICAL';
DELETE FROM configuration_language WHERE configuration_key = 'FACEBOOK_OPEN_GRAPH_STORED_URL';Dadurch entfernen Sie die nicht mehr benötigten Konfigurationseinstellungen, alle anderen unter Konfiguration > Facebook Open Graph bleiben erhalten.

Laden Sie nun die Dateien und Ordner aus NEUE DATEIEN im Ordner SCHRITT 2 - FACEBOOK OPEN GRAPH hoch (vorher den darin enthaltenen Ordner includes/DEINTEMPLATE auf den Namen des im Shop aktiven Templates umbenennen. Dabei werden bestehende alte Dateien des Moduls überschrieben, das ist ok.

Fertig, Sie haben auf 2.1 aktualisiert. Es sind keine Änderungen an den in SCHRITT 3 - FACEBOOK LIKEIT enthaltenen Dateien nötig.
