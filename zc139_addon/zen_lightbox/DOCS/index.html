Zen Lightbox 1.6.2 für Zen-Cart 1.3.9 deutsch

Übersicht | Installation

Sinn und Zweck dieses Moduls


Dieses Modul integriert den Lightbox Effekt in Zen-Cart. Alle großen Artikelbilder (auf der Detailseite) werden beim Anclicken in einer Lightbox angezeigt. Dadurch werden keine Popup Fenster mehr verwendet, was die Website wesentlich benutzerfreundlicher macht. Die zusätzlichen Artikelbilder werden innerhalb der Lightbox in einer Galerie angezeigt.

Mit Version 1.6.2 wird der Lightbox Effekt auf allen Seiten des Shops ermöglicht (auch auf EZ-Pages) und die Steuerung der Lightbox kann auch über Tastaturbefehle erfolgen. Damit die Lightbox auch in Firefox 4, Google Chrome und Internet Explorer 9 funktioniert, wird nun statt Mootools jQuery verwendet.
