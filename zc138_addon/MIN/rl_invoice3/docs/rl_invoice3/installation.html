Installation

1) Im Ordner includes/templates den Ordner YOURTEMPLATE auf das im Shop verwendete Template umbennen (z.B. viennablood, classic, ...)

2) Alle Dateien im Ordner includes in der vorgegebenen Struktur in die Zen-Cart Installation hochladen. Dabei wird folgende bestehende Datei überschrieben:

includes/classes/order.php

Sollten in dieser Datei bereits einmal Änderungen vorgenommen worden sein, vorher unbedingt den Inhalt vergleichen und die Änderungen manuell eintragen. Gut geeignet dazu ist das Tool Winmerge.

3) Alle Dateien im Ordner admin in der vorgegebenen Struktur in die Zen-Cart Installation hochladen. Dabei wird folgende Datei überschrieben:

admin/orders.php

Sollten in dieser Datei bereits einmal Änderungen vorgenommen worden sein, vorher unbedingt den Inhalt vergleichen und die Änderungen manuell eintragen. Gut geeignet dazu ist das Tool Winmerge.

4) Den Ordner pdf ins Hauptverzeichnis des Shops hochladen und diesem Ordner chmod 777 geben. In diesem Ordner werden später die pdf Rechnungen gespeichert, daher sind Schreibrechte nötig.

5) Wenn alle Dateien hochgeladen sind, in die Zen-Cart Administration einsteigen und Kunden > Bestellungen aufrufen.
Dort ist jetzt rechts bei jeder Bestellung ein neuer Button "PDF-Rechnung":

5) Bei irgendeiner Bestellung auf diesen Button clicken, dadurch werden die nötigen Einträge in der Datenbank vorgenommen und es geht der neue Menüpunkt Konfiguration > pdf Rechnung 3 auf. Befüllt ist die Konfiguration bereits mit funktionierenden Standardwerten. Diese Werte kann man dann später auf seine Bedürfnisse anpassen, mehr dazu unter "Konfiguration".

6) Die Installation ist damit abgeschlossen. Jetzt am besten mal das Erzeugen einer pdf Rechnung ausprobieren und nochmal unter Kunden > Bestellungen eine Bestellung auswählen und den Button "PDF-Rechnung" anclicken. Wenn dann ein pdf aufgeht mit dem Namen xxx.pdf (xxx=Bestellnummer) ist alles ok und das Modul ist einsatzbereit.

7) Jetzt kannst Du das Modul an Deine Bedürfnisse anpassen und bei "Konfiguration" weiterlesen.
