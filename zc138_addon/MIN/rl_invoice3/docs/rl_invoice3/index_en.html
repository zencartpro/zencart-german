pdf Invoice 3 for Zen-Cart 1.3.8

Overview | Installation | Configuration | FAQ

What does this module do?

pdf invoice 3 is a completely new module for Zen Cart 1.3.8.

An invoice in pdf format can be created at the touch of a button in the order overview area in Admin.
This invoice can optionally also be mailed straight away with the order and the customer has it later as a download in his/her order history.

Some features:

  • lexible adjustment of the layout is possible by using one's own pdf templates
  • Wide range of fonts included - can be extended according to your wishes
  • The pdf invoice can be automatically attached to the order confirmation mail when ordering
  • Offers the option of automatically sending documents about terms and conditions and about cancelling the order
  • The pdf invoice can be downloaded by the customer from their order history under 'My account'
  • Includes utf-8 support and therefore support for Zen Cart 1.3.8 German
  • Automated installation - no manual SQL import required
  • and much more...
