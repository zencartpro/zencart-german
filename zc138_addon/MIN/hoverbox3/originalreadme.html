Free Zen Cart Templates :: HoverBox3 Installation Readme

Excited!? Ready to get your HoverBox3 up and running? Let's start shall we...

  1. Upgrading? We've changed the structure and functionality of how HoverBox operates, so you'll need to manually remove a few files and directories that aren't used anymore. This will ensure smooth and proper operation. Let's do this first, then we'll move onto installing HoverBox 3.

    For reference: YOUR_TEMPLATE_FOLDER is the folder in which HoverBox was previously installed.

    Find and delete the following list of files and/or directories:
    • images/icpresent/
    • includes/functions/extra_functions/functions_hoverbox2.php
    • includes/templates/template_default/hoverbox/
    • includes/templates/YOUR_TEMPLATE_FOLDER/css/stylesheet_icimghover.css
  2. Un-pack the download package you recieved using WinZip or an archiving program of the like. Upgrading? ... Your safe to overwrite any existing files with the newer version. If your using Image Handler, HoverBox now detects if it's installed and runs accordingly.
  3. Upload the files in their existing structure, replacing YOUR_TEMPLATE_FOLDER with your currently active template folder.
  4. Next, to enable the administrative fuctions of HoverBox we need you to login to the administration area of your Zen Cart, when you've done that proceed!
  5. Now that your logged in, let's find "Tools" on the drop-down menu. Select "Install SQL Patches."
  6. Open the included sql file (use-sql-patch-tool-to-install.txt) from the sql folder in your favorite text editor. Copy & Paste the entire text file into "Enter Query to be executed."
  7. Ok, now select "Send." You should see a success message stating that the queries completed successfully.
  8. You're alllllllllllllll done! You have configuration options available to you under Configuration :: HoverBox 3. Enjoy!
