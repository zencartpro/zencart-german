简介：

在页面上显示在线帮助框, 目前支持Skype, Yahoo, 腾讯QQ, 淘宝旺旺.


---------------
安装
---------------

1. 修改template_default为你定义模版目录的名字
2. 上传includes目录到你商店的根目录覆盖
3. 进入管理后台 > 工具 > 安装SQL脚本 (livehelp.sql安装文件，uninstall.sql卸载文件)
4. 然后点 商店设置 > Live Help Configuration 进行配置
5. 完成

* 调整：工具 > 外观控制 调整sideboxes/live_help.php即可。

---------------
升级
---------------

1. 进入管理后台 > 工具 > 安装SQL脚本 > 导入uninstall.sql卸载先前的版本
2. 进行安装
3. 完成
